This plugin allows you to use Pay360 payment gateway within your WooCommerce.

== Description ==

WooCommerce Pay360 Payment Gateway will allows you to extend your WooCommerce installation with the Pay360 payment gateway.
The plugin integrates two types of Pay360 accounts Hosted and Hosted + IMA. In a few easy steps you will be ready to
accept Pay360 payments using WooCommerce.

This plugin was created by: vanbodevelops(Ivan Andreev). http://vanbodevelops.com

== Installation ==

1. Upload the `woocommerce-gateway-paypoint` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to `WooCommerce > Settings > Payment Gateways` to configure the Pay360 gateway settings.
